# Endfield Switcher (明日方舟：终末地账号切换器)

这是一个用 Rust (Tauri) + Vue 3 重构的开源本地工具，用于在 PC 端快速无缝切换《明日方舟：终末地》的账号。

## 💡 核心原理 (How it Works)

《明日方舟：终末地》的 PC 客户端会在本地电脑中存储您的免密码登录凭证文件（`login_cache`）。
本软件的底层逻辑非常简单、纯粹且安全，即：**本地文件替换**。

1. **备份记录**：软件会自动找到系统目录下当前的 `login_cache` 凭证，并复制一份存放到本软件自身的 `Backups` 文件夹里。
2. **切换游玩**：当您在列表中选中一个旧备份并点击「切换并启动」时，软件会自动把该备份的凭证**覆盖替换**回游戏目录，然后再启动游戏。

整个过程**纯本地且仅涉及文件复制粘贴**，不拦截网络、不修改内存，100% 安全防封号。

## ✨ 主要特点

- � **一键快照**：安全备份当前的登录状态，随意添加备注。
- 🔄 **智能切换与唤起**：选定账号，一键完成凭证替换与客户端启动。

- �️ **进程保护**：修改和备份前自动检查游戏是否在运行，杜绝数据损坏。

## 🚀 编译与使用

安装了 Node.js 与 Rust 环境后，您可以通过以下命令快速启动开发环境或构建发行版：

```bash
# 安装依赖
npm install
# 运行开发环境
npm run tauri dev
# 打包编译成 EXE
npm run tauri build
```
